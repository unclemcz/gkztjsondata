### 三、接口
由于参数中有中文，因此调用前接口地址需要进行URI编码
#### 1、获取试卷列表
https://www.wotiku.cn/apipro/getexamlist?key={key}&cls={cls}&index={pageindex}

key:授权码

cls:试卷类型，详见试卷范围部分

index:页码，从1开始，一次返回100行，调用的时候可以做循环，当返回的数据数量少于100行的时候，即表示已到最后1页

#### 2、获取具体一份试卷的内容&答案&解析
https://www.wotiku.cn/apipro/getexamdata?key={key}&examno={examno}&type={type}

key:授权码

examno：试卷自编号，第一个接口返回的试卷列表中可获取

type：试卷的组成部分，详见下表

试卷类型 | 大类-P |  试题-C | 答案-A | 解析-E
-|-|-|-|-
公务员-行测|√|√| |√
公务员-行测模拟|√|√| |√
公务员-申论| |√| | 
公务员-面试|√|√| |√
公务员-遴选|√|√| |√
事业单位-面试|√|√| |√
事业单位-公基|√|√| |√
事业单位-职测|√|√| |√
教师资格|√|√| |√
教师公开招聘|√|√| |√
特岗教师|√|√| |√
司法考试|√|√| |√

√打钩表示试卷类型对应一份试卷包含该部分，比如行测试卷由大类、试题与解析三部分组成，三部分组合成一份完整的试卷，申论只有试题部分，即试题部分已包含了一份申论试卷的全部内容，包括题本和解析；一般来说大类指的是一份试卷各部分介绍、试题指的是每个题目内容和答案、解析指的是对应每道题的解析。
